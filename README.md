# Как развернуть проект

## Создание виртуальное окружение
```bash
python -m venv venv
```

## Активация вирутального окружения

### Для Windows:
```bash
.\venv\Scripts\activate
```

### Для Linux и MacOS:
```bash
source venv/bin/activate
```

## Чтобы скачать зависимости в venv
```bash
pip install -r .\requirements.txt
```
